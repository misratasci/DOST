# Plot DOST Spatial Clusters

Visualizes the clustering results in spatial coordinates.

## Usage

``` r
plot_DOST_spatial(coords, labels, point_size = 1)
```

## Arguments

- coords:

  A data frame or matrix of spatial coordinates (Spots x 2).

- labels:

  A vector of cluster labels.

- point_size:

  Numeric, size of the points (default 1).

## Value

A ggplot object.
